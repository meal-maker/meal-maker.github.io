## [Han: A Korean Programming Language Written in Rust](https://github.com/xodn348/han) ⭐️ 7.0/10

A developer has created Han, a statically-typed Korean programming language written in Rust with AI assistance. It features a full compiler pipeline from lexer to LLVM IR code generation and supports modern language features such as pattern matching, closures, and a basic LSP server. This project demonstrates how AI can accelerate language development and makes programming more accessible to Korean speakers by using Hangul keywords. It contributes to linguistic diversity in tech and explores how natural language integration might lower learning barriers for non-English speakers. Han includes a comprehensive compiler architecture with lexer, parser, AST, interpreter, and LLVM IR codegen, supporting arrays, structs with impl blocks, closures, pattern matching, try/catch, file I/O, module imports, a REPL, and a basic LSP server. However, it is a side project and not intended for production use.

hackernews · xodn348 · Mar 14, 21:27

**Background**: Hangul is the alphabetic writing system used for the Korean language, known for its logical and easy-to-learn design. LLVM IR (Intermediate Representation) is a low-level code representation used in compilers to enable optimizations and code generation for various hardware targets. The Language Server Protocol (LSP) is a standardized protocol that allows development tools to communicate with language servers for features like auto-completion and go-to-definition.
