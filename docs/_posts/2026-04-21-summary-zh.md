## [量子计算机对 128 位对称密钥不构成威胁](https://words.filippo.io/128-bits/) ⭐️ 8.0/10

Filippo Valsorda 的文章阐明，由于 Grover 算法的限制，量子计算机对 128 位对称密钥不构成实际威胁，因为破解此类密钥需要不可行的 2^64 次操作。 这一澄清非常重要，因为它消除了关于量子计算将对称密钥安全性减半的普遍误解，使组织能够对现有加密标准（如 AES-128）保持信心，而无需紧急升级。 Grover 算法提供二次加速，将 128 位密钥的暴力搜索复杂度从 O(2^128) 降低到 O(2^64)，但以当前量子技术，这仍然是计算上不可行的。文章强调，作为后量子密码学迁移的一部分，对称密钥大小无需增加。

hackernews · hasheddan · Apr 20, 16:37

**背景**: 对称密钥密码学使用相同的密钥进行加密和解密，其安全性取决于密钥大小。Grover 算法是一种量子搜索算法，可以比经典方法二次更快地在未排序数据库中找到解决方案，从而有效降低对称密钥的安全级别。然而，对于 128 位密钥，由于巨大的计算需求，降低到 64 位的量子安全性仍被认为是坚固的。

**背景**: 图形语法是一种基于语法的图形表示系统，由 R 中的 ggplot2 等工具推广，它将可视化分解为数据、美学和几何体等组件。SQL 是用于查询和管理关系数据库的标准语言，在数据分析中广泛使用。ggsql 将两者结合，允许在 SQL 查询中指定可视化。

## [Firefox WebUSB 扩展实现网页应用跨浏览器访问 USB 设备](https://github.com/ArcaneNibble/awawausb) ⭐️ 7.0/10

一个名为 'awawausb' 的 Firefox WebUSB 扩展使网页应用能够直接从浏览器访问 USB 设备，解决了 Firefox 缺乏原生 WebUSB 支持的问题。该扩展提供了跨浏览器兼容性，使得无需依赖 Chrome 即可实现硬件集成。 这很重要，因为它减少了对 Chrome 在 WebUSB 硬件访问上的依赖，促进跨浏览器兼容性，并通过允许开发者创建在 Firefox 上也能运行的硬件集成网页应用来支持开放网络生态系统。它有助于避免浏览器垄断，并推动网络标准的采用。 该扩展以浏览器插件形式实现 WebUSB API，需要用户为每个 USB 设备明确授予访问权限，类似于原生实现。然而，作为扩展，它在性能、安全或集成方面可能不如内置的浏览器支持。

hackernews · tuananh · Apr 20, 11:51

**背景**: WebUSB 是一个 JavaScript API 规范，可安全地让网页应用与 USB 设备通信，实现无需安装原生应用的硬件交互。虽然 Chrome 原生支持 WebUSB，但 Firefox 一直拒绝实现，导致依赖 USB 访问的网页应用存在兼容性问题。该 API 是通过硬件集成增强网络能力更广泛努力的一部分，需要用户交互（如选择设备）以确保安全。
